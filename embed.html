How to Embed This Calculator

Basic Embed Code

Copy and paste this HTML code into your website:

<iframe src="https://yourusername.github.io/calculator-base/" width="100%" height="800px" frameborder="0" title="Wallpaper Calculator"> </iframe>

Responsive Embed

For a responsive design that adapts to different screen sizes:

<div style="position: relative; width: 100%; height: 0; padding-bottom: 75%;"> <iframe src="https://yourusername.github.io/calculator-base/" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;" frameborder="0" title="Wallpaper Calculator"> </iframe> </div>

Calculator Features

  • Pattern selection from CSV data
  • Wall dimension input (feet and inches)
  • Visual preview generation
  • Panel and yard pattern support
  • PDF generation and download
  • Quote request forms
  • Mobile responsive design

Setup Requirements

  • GitHub account for hosting
  • CSV file with pattern data
  • Pattern images (JPG/PNG)
  • Configuration file customization
  • Optional: Google Sheets integration
  • Optional: reCAPTCHA setup

Customization Options

  • Business information and branding
  • Color scheme and styling
  • Text content and messaging
  • PDF layout and design
  • Form fields and requirements
  • Calculation defaults and limits

Browser Support

  • Mobile browsers

Getting Started

To create your own calculator:

  1. Fork the calculator-base repository
  2. Update the config.js file with your business information
  3. Replace data/patterns.csv with your pattern data
  4. Add your pattern images to the data/patterns/ folder
  5. Enable GitHub Pages in your repository settings
  6. Embed the calculator on your website using the iframe code above

For detailed instructions, see the README documentation.
